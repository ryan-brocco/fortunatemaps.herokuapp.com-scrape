# Scrape of fortunatemaps.herokuapp.com
The purpose of this scrape is to allow for more robust searching for TagPro maps hosted on fortunatemaps.herokuapp.com than what is possible through the site's native search function or through a web search for site:fortunatemaps.herokuapp.com. The native search function on fortunatemaps.herokuapp.com is limited. For example, there is currently no way to search for unlisted maps or for maps by game mode (e.g., normal, gravity, etc.). The files in this repository can be downloaded and opened with text editors that have enhanced search functionality.
<br>

About each file
---
\- FM maps (map IDs 1-88850).xlsx<br>
\- scrape results (map IDs 1-88850).txt<br>
\- scrape results - JSONs only (map IDs 1-88850).ndjson<br>
<br>
\- One line per map ID<br><br>
